## ☃️What

使用`golang`编写的ThinkPHP扫描器，`POC`采用`ymal`文件加载

## 🤪How

- 单个`URL`测试

```bash
./TpScan.exe -u http://node4.buuoj.cn:26433/
```

- 批量`URL`测试

```bash
./TpScan.exe -f test.txt
```
